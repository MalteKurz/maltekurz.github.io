Blog Articles on Quant’s Perspective

A Python Implementation of Triangles for Visualising Long-Term Investment Metrics

Visualisation techniques are a key element of any investment analysis. We provide an open-source python implementation of triangle plots for visualising long-term investment metrics.

Trade Cost Optimisation II: Tracking Error and the Cutting Plane Algorithm

Measuring the distance between two portfolios can be achieved with different metrics, like turnover or tracking error. We show how a tracking-error-based trade cost optimisation can be implemented using the cutting plane algorithm to solve mixed integer quadratic programs (MIQPs).

Trade Cost Optimisation

Minimising trading costs and enforcing a no-fractional-dealing condition are two major challenges when implementing dynamic portfolio strategies. We present a flexible and efficient trade cost optimisation algorithm utilising a classical mixed integer linear program (MILP).
